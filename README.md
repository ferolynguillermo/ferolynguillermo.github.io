# Google Profile Site

Edit `site.config.js` first. Replace `https://example.com/` in `site.config.js`, `robots.txt`, and `sitemap.xml` with your live website URL before publishing.

For Google search visibility:

1. Publish the folder with a public URL.
2. Replace every placeholder with real profile information.
3. Use your full name consistently in the page title, heading, bio, and profile links.
4. Submit the live URL and sitemap in Google Search Console.

Google decides whether and when a page appears in search results, so indexing is not instant or guaranteed.
